# Liver Disease Prediction Using Blood Reports

## Overview
This project aims to predict liver disease based on blood reports using machine learning. The project includes a Flask-based web application for interacting with the prediction model and displaying results to users.

## Project Structure
- app.py: Flask application script for handling web requests and rendering the frontend.
- model.py: Python script containing the machine learning model implementation and prediction logic.
- templates: Directory containing HTML templates for the frontend.
- model.pkl: Trained machine learning model serialized as a .pkl file for easy loading and prediction.

## Requirements
- Python 3.x
- Flask
- Scikit-learn
- NumPy
- Pandas
